Some very useful open-source libraries in C & C++

  • Boost: One of the very powerful open-source libraries in C++. It consists of very efficient libraries for Arrays, Algorithms, Geometry, Graphs, Math functions, Timers etc.
  • Eigen: A very efficient Matrix library built purely based on Expression Templates. It does not require compilation.
  • PETSc: A parallel Matrix Algebra library with numerous parallel iterative solvers and support for many third-party solver libraries.
  • Intel MKL: Matrix library from Intel. (You have to purchase it!)
  • VTK: The best open-source library (in my opinion) for geometry processing and (FEM/CFD) data visualisation.
  • CGAL: A powerful library for Geometry algorithms.
